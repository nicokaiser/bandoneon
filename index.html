How to use

To internalize the button location on the both manuals the button map can be displayed.

  • The tabs above the keyboard switch the direction of the bellows (open/pull or close/push) and the side (left or right).
  • Select a key and a mode to display a scale — e.g. C Major.
  • Select a key and a quality to display a chord — e.g. E m7.
  • Keyboard shortcuts: l, L, r, R select the side and direction; M, m, 7 select Major, Minor and Septime chord; c, d, e, f, g, a, b select the key; add # to select sharp key; C toggles button colors.

The keyboard layout is based on the Alfred Arnold 142 button Bandoneon (bisonoric, “Rheinische Tonlage”)
